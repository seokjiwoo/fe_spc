  • 단일/묶음 배송템플릿 설정여부에 따라 노출되는 화면이 상이합니다.
  • 패널티는 딜의 주문 기준으로 확인 가능합니다.
  • 예정 패널티 금액을 클릭하면 패널티 금액에 대한 소명이 가능합니다. 단, 소명처리상태가 '승인대기' 혹은 '미접수' 일 때만 가능합니다.

  • 패널티 금액에 대한 소명가능기간은 정산방식 개편 전후로 다르게 적용됩니다.

    - 개편 전 상품: 지연기준일 + 10배송영업일 ~ 17배송영업일까지

    - 개편 후(입점계약 및 전달계약) 상품: 지연기준일부터 지연기준일 + 60일까지입니다.

    단, 지연기준일 +10배송영업일 전에 소명할 경우 배송상태가 배송시작 혹은 배송완료여야 소명가능합니다. (지연기준일: 배송이 지연되기 시작한 일자)

  • 좌측의 선택박스를 여러 개 클릭하면 한 번에 여러 개의 패널티 금액에 대해 소명도 가능합니다.
  • 소명접수를 하시면 티몬에서 접수를 받아 소명접수를 승인하거나 거절하게 됩니다.
  • 정산액에 반영될 금액은 확정패널티 금액으로 확인하시면 됩니다.
