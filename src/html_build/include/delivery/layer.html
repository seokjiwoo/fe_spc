배송관리 도움말

  1. 조회조건: 미확인 주문 선택하여 조회하세요.
  2. [주문확인] 버튼 클릭 시 주문내역 엑셀 다운로드와 동시에 상태가 송장등록대기 상태로 변경됩니다.(매일 오후 3시까지 주문된 내역을 확인해주세요!)
  3. 조회조건에서 송장등록대기 상태를 선택 후 [엑셀 다운로드]
  4. 엑셀 파일 C열 택배사와 D열 운송장번호 입력, 저장하세요.
  5. [송장 업로드] 버튼을 클릭하여 업로드하시면 송장등록완료 상태로 변경됩니다.

운송장 번호 중복사용에 대한 안내

  • 이미 업로드 되어 있는 송장을 수정할 수 있으나 고객에게 '합배송'을 하기 위한 목적 이외에, 기존에 등록된 운송장 번호와 동일한 운송장번호를 등록 하는 경우에는 정산 상 불이익이 갈 수 있습니다.
  • 운송장번호 등록과 관련한 문의 사항이 있으신 경우 스토어 파트너 컨택 센터[1644-0552]를 통해 안내 받으실 수 있습니다.

배송지연 사전신고 및 재고품절 처리 안내

상품 판매 중 예상치 못한 배송지연이나, 더 이상 재고 수급이 어려울 경우
다음과 같이 처리 하실 수 있습니다.

  • 01. 배송이 지연될 것으로 예상되는 주문을 검색해주세요.
  • 02. 주문을 선택한 후 배송지연 신고 버튼을 눌러주세요.
  • 03. 지연/상세 사유를 입력하고 등록하면 끝!
  • ※작성하신 내용은 해당 고객에게 안내되며, 출고예정일 이전에 미리 신고하시면 지연으로 인한 페널티 금액을
    경감받으실 수 있습니다.
  • 01. 품절(판매중지)처리 : 처리 전 꼭! 담당 MD와 먼저 상의 판매가 불가한 옵션을
    선택하신 후 등록하면, 즉시 품절상태로 변경됩니다.
  • 02. 품절로 인한 환불처리 : 구매된 주문 중 부득이하게 환불처리가 필요한 경우
    재고품절 사유로 환불 등록 후 승인 하시면, 옵션 판매가의 10%가 해당 고객님께
    적립금으로 지급됩니다.
  • ※해당되는 고객님께 꼭 사전에 연락하셔서 양해를 구해주세요.
  • ※출고예정일 후에 품절 등록하실 경우 정산시 패널티 금액이 발생할 수 있으니 재고관리에 유의해 주세요.
